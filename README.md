# AI Text Generation API

A simple AI-powered text generation API built using FastAPI and Hugging Face Transformers. The application uses Google's FLAN-T5 model to generate text responses from user input and is containerized using Docker for deployment on Hugging Face Spaces.

## 🚀 Features

- REST API built with FastAPI
- AI-powered text generation using Hugging Face Transformers
- Dockerized application for easy deployment
- Deployable on Hugging Face Spaces

## 🛠️ Tech Stack

- Python 3.9
- FastAPI
- Transformers
- PyTorch
- Docker
- Uvicorn
